# Hybrid Search Strategy

Use this reference when deciding how to interpret or follow up on merged results.

## Provider roles

- SearchCans: good for fast URL discovery and broad SERP-style coverage
- Linkup: useful as a complementary provider with different recall characteristics

## Practical use

- Start with both providers when both keys are available
- Treat the merged set as a candidate URL pool
- Fetch or read only the most relevant URLs afterward

## Deduplication priorities

- Exact URL match first
- Near-duplicate title match second
- Prefer the earlier provider in the configured priority order when duplicates collide
