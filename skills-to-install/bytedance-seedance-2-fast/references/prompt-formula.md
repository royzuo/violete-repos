# Prompt Formula

Use this reference when the user gives a topic instead of a finished prompt.

## Core formula

```text
[Camera] + [Subject] + [Action] + [Atmosphere] + [Lighting] + [Quality Tags]
```

## What to include

- `Camera`: movement, framing, lens feel
- `Subject`: the central object, person, or scene
- `Action`: what changes or happens in the shot
- `Atmosphere`: environment, weather, tone, era
- `Lighting`: time of day, contrast, mood, color temperature
- `Quality Tags`: short stylistic qualifiers, not resolution spam

## Good defaults

- Prefer one coherent shot over many disconnected ideas
- Keep the prompt visually concrete
- Avoid generic adjectives with no image value
- Avoid unnecessary resolution tags
