## 核心问题

- 洞里萨河为什么会在一年中的某个阶段倒着流？
- 洞里萨湖为什么能在几个月内从一个较小湖盆膨胀成巨大的季节性内陆水域？
- 这种“逆流—扩湖—退水”的节律，为什么会变成东南亚最重要的内陆渔业与粮食系统之一？
- 当上游水坝、气候异常和鱼类下滑同时出现时，被冲击的首先是什么？

## 核心结论

- 洞里萨湖的“逆流”不是偶然奇观，而是湄公河主河道与洞里萨湖之间高程差和季风洪峰共同驱动的季节性回灌机制。
- 每年雨季，湄公河洪水在金边附近抬高水位，迫使原本向南泄水的洞里萨河反向北流，把洪水顶回洞里萨湖，使湖面和水量迅速扩张。
- 到旱季，湄公河水位回落，洞里萨河再次转向，把湖中的蓄水慢慢排回湄公河，因此洞里萨湖本质上是一个会“呼吸”的季风调蓄盆地。
- 正是这套洪泛节律，创造了洪泛森林、鱼类繁殖栖息地、浮村生活和世界级内陆渔业；洞里萨不是单独一片湖，而是区域粮食系统的重要缓冲器。
- 真正的风险并非单一“少水”，而是洪峰时间、逆流规模、淹水持续期和泥沙—鱼类—森林之间的耦合关系被改写；当节律被打乱，渔业、生计和生态会一起下滑。

## 证据链

1. 证据 / 数据 / 论文 / 报告：Britannica Tonle Sap 条目
   - 支撑什么结论：季风期湄公河洪水可逆转洞里萨河流向，使湖面从约 2700 平方千米扩张到约 10360 平方千米
   - 不确定点：面积和体积会因年份不同而变化，节目中应以“约”“可达”表述
2. 证据 / 数据 / 论文 / 报告：Mekong River Commission 程序与管理材料
   - 支撑什么结论：逆流是湄公河流域独特现象，雨季会把过量洪水推入湖中，湖体可扩张为常态的数倍
   - 不确定点：MRC 不同材料采用“四倍”“六倍”等不同表达，节目中强调“数倍扩张”更稳妥
3. 证据 / 数据 / 论文 / 报告：UNESCO Tonle Sap Biosphere Reserve 材料
   - 支撑什么结论：洞里萨湖与湄公河年度洪泛脉冲紧密相连，生物多样性和社区生计依赖这种洪泛节律
   - 不确定点：保护区尺度和整个湖区尺度不同，节目中应区分生态核心区与整体湖区
4. 证据 / 数据 / 论文 / 报告：WCS / National Geographic 等保护与科普材料
   - 支撑什么结论：季节性扩湖支撑洪泛栖息地和巨大的渔业生产，一旦水位偏低，栖息地和鱼类会显著承压
   - 不确定点：具体鱼产量年份波动大，不宜使用单一年份数字做绝对判断
5. 证据 / 数据 / 论文 / 报告：MRC 与近年新闻摘要（2024-2025）
   - 支撑什么结论：逆流趋势减弱、鱼类资源下滑和低水位问题仍在引发流域层面的担忧
   - 不确定点：成因涉及水坝、气候异常、土地利用变化等多因素，节目中避免单因归责
6. 证据 / 数据 / 论文 / 报告：FAO / 历史渔业资料
   - 支撑什么结论：洞里萨渔业和水上生活与河湖节律、航道、洪泛森林密切相关
   - 不确定点：旧资料能支撑结构逻辑，但现状部分需结合近期材料谨慎更新

## 现状、挑战与未解之谜

- 挑战一：上游水坝与流域调度可能改变洪峰到达时间、峰值高度和泥沙通量
- 挑战二：气候异常使丰枯节律更不稳定，低水位年份会直接压缩鱼类繁殖空间和洪泛森林补给
- 挑战三：渔业资源下滑与社区生计脆弱化同步出现，浮村和湖边聚落对节律变化高度敏感
- 挑战四：洞里萨是湖泊、河流、森林和粮食系统的耦合体，保护不能只盯某一种鱼或某一段河道
- 未解之谜 / 科学持续议题：逆流规模的长期趋势、洪泛森林恢复阈值、鱼类迁徙对流量与水质变化的敏感度、泥沙减少对湖盆生态的复合效应，仍在持续研究中
